# Domain Docs — 单上下文模式

## 文档结构

```
/
  CONTEXT.md          # 根级上下文文件，包含项目全局背景
  docs/
    adr/              # 架构决策记录（Architecture Decision Records）
    agents/           # Agent 配置文档（本目录）
```

## 约定

- **`CONTEXT.md`** 维护项目全局背景、当前状态、关键决策
- **`docs/adr/`** 存放结构化架构决策记录，每个 ADR 一个文件
- 技能按需创建文档，**不强制预创建**
