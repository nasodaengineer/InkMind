# Triage Labels

默认五角色标签体系（保留默认名称）：

| 标签 | 说明 | 职责 |
|------|------|------|
| `needs-triage` | 待分类 | 新提交的问题/任务，尚未审查 |
| `needs-info` | 待补充信息 | 缺乏复现步骤或上下文，需提问者补充 |
| `ready-for-agent` | 可分配给 Agent 处理 | 信息完整，可由自动化 Agent 处理 |
| `ready-for-human` | 需人工介入 | 需要人类开发者/写手决策或处理 |
| `wontfix` | 不予处理 | 经评估后决定不处理的问题 |
